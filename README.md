# 📊 Global Terrorism Dataset – Exploratory Data Analysis

Exploratory Data Analysis on the Global Terrorism dataset (1970–2017) examining 180,000+ records. Built using Pandas, Matplotlib & Seaborn to highlight attack trends, country risk levels, and weapon patterns. Insights help stakeholders understand long-term historical patterns and regional risk profiling.

## 📌 Overview
This repository contains an **Exploratory Data Analysis (EDA)** on the Global Terrorism Dataset (1970–2017).  
The analysis uncovers historical attack patterns, high-risk regions, attack types, and trends using visualizations and Python.

📅 **Project completed:** January 2025  
🔄 **Last updated:** March 2025

---

## 🔍 Key Insights
- Analyzed **180,000+ records** over multiple decades  
- Identified **country-wise attack trends**  
- Explored **weapon types, attack frequency, and regions**  
- Visualized insights with **Matplotlib & Seaborn**
